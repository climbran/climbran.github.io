三、常用容器
(1)List
ArrayList 非线程安全，能快速进行随机访问
LinkedList 非线程安全，底层是双向链表，增删比ArrayList快，能更好的支持栈，同时实现了Queue接口，能支持队列操作。
Vector 相当于线程安全版ArrayList,如果不考虑并发，ArrayList效率较高，当数组长度不够时，Vector增加100%，ArrayList增加50%

(2)Map
HashMap 非线程安全，基于散列表实现，时间开销为O(1)，一般效率比TreeMap 非线程安全，基于红黑树实现，时间开销为O(logN),效率比HashMap低，但内部是有序的，TreeMap是唯一带有subMap()方法的Map，可以返回一个子树
ConcurrentHashMap 线程安全，实现ConcurrentMap接口，根据hash值分段加锁。
